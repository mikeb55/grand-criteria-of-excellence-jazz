# **Crystal Silence — Tune #11**

| Key | Form | Time | Bars | Style | Composer |
|:---:|:---:|:---:|:---:|:---|:---|
| A Major | 16-bar form | 4/4 | 16 | ECM Ballad (Ralph Towner / Egberto Gismonti) | Chick Corea |

**Uniqueness Tagline:** Bell-like crystalline textures with open strings ringing like a 12-string harp.  
**Playing Tip:** Never dampen strings early — let the reverb tail become part of the composition.  
**Analysis Focus:** Open-string pedal points and campanella voice leading.  
**Practice Priority:** Campanella technique — adjacent notes on different strings for maximum sustain.

## **5. TWO-VOICE COUNTERPOINT BREAKDOWN (Wyble)**

### Bars 1–4: Campanella Parallel/Similar Motion
```
Top Line:    C#5 → A4 → F#4 → E4
Bottom Line: A2 (pedal) → A2 → A2 → A2
Motion:      Oblique — melody moves over static bass pedal
Comment:     Open A string sustains through all four bars; melody descends in thirds.
```

### Bars 5–8: Contrary Motion to Cadence
```
Top Line:    C#5 → E5 → D5 → G#4 → G#4
Bottom Line: A2 → C#3 → B2 → E3 → E3
Motion:      Contrary — bass rises as melody falls toward cadence
Comment:     Classic ii-V-I voice leading with suspension resolution.
```

### Bars 9–10: Modal Interchange (Distant Colors)
```
Top Line:    E5 → D5
Bottom Line: F3 → G3
Motion:      Parallel — both voices rise together
Comment:     bVI → bVII creates chromatic ascending bass; borrowed from parallel minor.
```

### Bars 11–16: Resolution
```
Top Line:    C#5 → F#4 → D5 → G#4 → C#5
Bottom Line: A2 → D3 → B2 → E3 → A2
Motion:      Mixed — contrary then parallel resolution
Comment:     Final cadential motion returns to open A pedal; campanella texture maintained.
```

## **REFERENCES**

**Influences:**
- Chick Corea & Gary Burton duo recordings
- Ralph Towner (Oregon, solo guitar)
- Egberto Gismonti (ECM recordings)
- Pat Metheny early ECM work

**ECM Aesthetic Characteristics:**
- Space as composition — reverb tail is part of the music
- Quiet dynamics — never louder than mezzo-piano
- Open voicings — wide intervals between notes
- Melodic simplicity — fewer notes, more sustain
- Timbral focus — tone quality over velocity
