# ğŸ”‘ CORE CONCEPTS

### 1. Melody-Derived Harmony (MDH)
Chords exist to support melody. Harmony is generated from melodic content.

### 2. Triad-Pair System
- **Stable Triad**: Contains chord tones (1, 3, 5)
- **Color Triad**: Adds tensions (7, 9, 11, 13)

### 3. Jimmy Wyble Counterpoint
Independent two-voice writing:
- Top voice: Melody
- Bottom voice: Counter-melody

### 4. Barry Harris Logic
6th Diminished scales for smooth bebop voice leading.

### 5. Tonality Vault
Harmonic universes for each chord type.
