# THE CONCEPT

Blue Cycle takes the traditional 12-bar blues and infuses it with two sophisticated elements:

1. **Jimmy Wyble Counterpoint:** Two independent voices on one guitar
2. **Coltrane Changes:** Cycle-based chord substitutions

The result is a modern blues that honors the tradition while pushing into contemporary territory.

# COUNTERPOINT GUIDE

## Wyble Principles Applied

1. **Voice Independence:** Each voice has its own melodic logic
2. **Contrary Motion:** 58% of the time, voices move in opposite directions
3. **Divergence-Convergence:** Expand to climax (bar 6), contract to resolution (bar 12)
4. **Fingering Awareness:** Consider string sets for playability

## Key Technical Moments

### Bar 6 — The "Wyble Moment"
```
TOP:    Bb whole note (sustained high) ─────────────
BOTTOM: E → G → G# → A (chromatic quarter-note walk)
```
This is the peak of tension — M7 interval between voices.

### Fingering Zones
- Bars 1-4: Position V (home base)
- Bars 5-8: Position VII (climb)
- Bars 9-12: Return to V (resolution)
