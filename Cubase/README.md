# Cubase Projects

This directory contains Cubase project templates and session files for the GCE Jazz workflow.

## Documentation & Prompts

* **`Cubase_RC_TheScore_Template_Prompt.md`**: A master instruction prompt for configuring the integration between Cubase, Rapid Composer, and Sonuscore's "The Score".
    *   **Purpose:** Use this prompt with AI assistants (Cursor, ChatGPT) or as a manual checklist to set up the complex routing required for the GCE Jazz workflow.
    *   **Scope:** Covers track creation, MIDI routing, instrument loading, and sync settings.
