# Rapid Composer Workspace

This directory contains assets, presets, and MIDI files compatible with Rapid Composer (RC).

## Contents

### Template Files
* **`RC BLANK Orchestral TEMPLATE.rcCOMP`**: A clean, pre-configured Rapid Composer composition template set up for the Small Jazz Orchestra. It includes the correct track routing for:
    *   Winds (Flute, Clarinet, Flugelhorn)
    *   Strings (Vln I, Vln II, Viola, Cello)
    *   Rhythm (Guitar, Piano, Bass, Percussion)
  Use this as a starting point for new GCE Jazz compositions to ensuring seamless MIDI export to "The Score".

### Other Assets
* **Presets:** GCE Jazz Engines A, B, C, D.
* **Livesets:** Configuration files for RC Live Mode.
* **MIDI:** Multi-track MIDI exports for The Score integration.
